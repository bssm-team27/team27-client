# 해양 안전 선택지 기반 시뮬레이션 사이트 PRD

## 1. 프로젝트 목표 및 범위

### 1.1 프로젝트 목표
- 스토리 시뮬레이션 기반의 선택지 게임을 통해 해양 안전 사고 예방 교육 제공
- AI 기반 실시간 시나리오 및 배경 이미지 생성으로 몰입도 높은 교육 경험 구현
- 직관적이고 접근성 높은 웹 애플리케이션 개발

### 1.2 프로젝트 범위
**포함 기능:**
- 게임 시작 전 설정 (인원, 활동, 날씨, 상황)
- AI 기반 시나리오 및 배경 이미지 생성
- 선택지 기반 게임 진행 및 즉시 피드백
- 게임 종료 후 분석 및 피드백 제공
- 게임 히스토리 저장

**제외 기능:**
- 사용자 인증/인가 시스템
- 게임 진행 상태 저장/복원
- 회원 관리 기능

## 2. 사용자 플로우 다이어그램 (텍스트 설명)

```
1. 메인 페이지 진입
   ↓
2. "게임 시작" 버튼 클릭
   ↓
3. 게임 설정 페이지
   - 인원 선택 (1인/2인/단체)
   - 활동 선택 (해수욕/낚시/레저활동)
   ↓
4. "게임 생성" 버튼 클릭 → 게임 생성 API 호출
   ↓
5. 인게임 페이지
   - 배경 이미지 표시
   - 상황 설명 및 선택지 제시
   - 선택지 선택 → 선택지 응답 API 호출
   - 즉시 피드백 표시
   - 다음 상황으로 진행 (반복)
   ↓
6. 게임 종료 후 분석 페이지
   - 분석 API 호출
   - 전체 선택에 대한 분석 및 피드백 표시
   ↓
7. 새로운 게임 시작 또는 종료
```

## 3. 주요 화면별 상세 명세

### 3.1 메인 페이지 (MainPage)

**UI/UX 요구사항:**
- 서비스 소개 헤더
- "게임 시작" 버튼
- 해양 테마의 배경 디자인
- 반응형 레이아웃 지원

### 3.2 게임 설정 페이지 (GameSetupPage)

**UI/UX 요구사항:**
- 인원 선택 라디오 버튼 그룹
- 활동 선택 카드형 UI
- "게임 생성" 버튼 (설정 완료 시 활성화)
- 로딩 스피너 (게임 생성 중)

### 3.3 인게임 페이지 (GameplayPage)

**하위 컴포넌트:**
- `BackgroundImage`: AI 생성 배경 이미지 표시
- `ScenarioDescription`: 상황 설명 텍스트
- `ChoiceList`: 선택지 목록

**UI/UX 요구사항:**
- 풀스크린 배경 이미지
- 상황 설명 오버레이
- 선택지 버튼들 (하단 고정)
- 화면 클릭 시 다음 대사로 넘어감

### 3.4 분석 페이지 (AnalysisPage)

**UI/UX 요구사항:**
- 점수 및 성과 요약
- 선택별 영향 그래프
- 개선점 및 강점 리스트
- "새 게임 시작" 버튼

## 4. 기술 스택 및 아키텍처

### 4.1 기술 스택
- **Frontend Framework**: React 19+ with TypeScript
- **State Management**: Zustand
- **Styling**: Tailwind CSS
- **HTTP Client**: Axios
- **Build Tool**: Vite
- **Package Manager**: npm

### 4.2 아키텍처

**폴더 구조:**
```
src/
├── components/           # 재사용 가능한 UI 컴포넌트
│   ├── ui/              # 기본 UI 컴포넌트 (Button, Modal 등)
│   ├── game/            # 게임 관련 컴포넌트
│   └── common/          # 공통 컴포넌트
├── pages/               # 페이지 컴포넌트
│   ├── MainPage.tsx
│   ├── GameSetupPage.tsx
│   ├── GameplayPage.tsx
│   └── AnalysisPage.tsx
├── stores/              # Zustand 스토어
│   ├── gameStore.ts
│   └── uiStore.ts
├── api/            # API 서비스
│   └── gameAPI.ts
├── types/               # TypeScript 타입 정의
│   └── game.ts
└── App.tsx
```
