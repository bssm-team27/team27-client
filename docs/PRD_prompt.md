## 프로젝트 개요
### 서비스명: 해양 안전 선택지 기반의 시뮬레이션 사이트
### 핵심 가치: 스토리 시뮬레이션 기반의 선택지 게임을 통해 사용자가 적절한 선택지를 고르면서 자연스럽게 해양 안전 사고를 예방하도록 돕는 교육 서비스
AI 활용(백엔드 역할):
사용자의 요구 상황에 맞게 시나리오와 배경을 실시간으로 생성

## 게임 시작 전 선택사항
- 인원 선택: 1인, 2인, 단체(3인 이상)
- 활동 선택: 해수욕, 낚시, 레저활동
- 날씨: 랜덤 생성
- 상황: AI 기반 랜덤 생성

## 게임 진행 방식

- 각 상황마다 2개 이상의 선택지 제공
- 선택 즉시 긍정/부정 피드백 표시(백엔드에서 받아오는 값)
- 게임 종료 후 전체 선택에 대한 분석 제공

## 화면 구성

### 메인 페이지: 게임 시작 화면
### 게임 시작 전 선택 페이지: 인원, 활동 선택 → 게임 생성 API 호출
### 인게임 페이지:
- 배경 이미지 표시 (AI 실시간 생성)
- 상황 설명 대사
- 선택지 UI
- 즉각 피드백 표시


### 분석 페이지: 게임 결과 분석 API 호출 및 결과 표시

**API 연동 요구사항**

- 게임 생성 API: 설정 기반 시나리오 생성
- 선택지 응답 API: 사용자 선택에 따른 다음 상황 제공
- 배경 생성 API: AI 기반 실시간 배경 이미지 생성
- 분석 API: 게임 결과 분석 및 피드백 제공

**제외 기능**

- 사용자 인증/인가
- 게임 진행 상태 저장
- 회원 관리 기능
